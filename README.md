# Boilerplate: VueJS 2 + ASP.NET MVC 5

Integrating Vue.js 2.0 to ASP.NET MVC 5 Project

Steps:

```
cd VueJS2-MVC5
npm install
npm run build
```
Open the solution on Visual Studio and run the web project. Make sure to run the web project on http://localhost:54490

For hot-reaload

```
npm run dev
```
